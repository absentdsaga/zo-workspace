# Copy Collaboration Hub

This skill orchestrates multiple experts to collaborate on creating exceptional copy. Use this for any significant copy task.

## How It Works

When you need to create copy, activate these perspectives in order:

### 1. Solana Mobile Expert (First)
- What pain points do Solana Mobile builders actually face?
- What terminology resonates in this community?
- What's the ecosystem-specific value?
- What do they already know vs what needs explaining?

### 2. Product Positioning Expert (Second)
- What is the one thing only this product offers?
- What job is the customer hiring this product to do?
- What is the core transformation, not just features?
- How does this beat every alternative?

### 3. B2B SaaS Copywriter (Third)
- What headline formula applies?
- How do we structure for 8-second conversion?
- What's the pricing psychology?
- What CTAs convert?

### 4. Web3 Native Marketer (Fourth)
- Is this voice crypto-native?
- Are the proof points on-chain verifiable?
- Does this feel "from the community" or "corporate trying to be crypto"?
- What triggers skepticism vs trust?

## Collaboration Process

For each copy task:

1. **Gather** — Have each expert answer the same question from their lens
2. **Synthesize** — Find where insights overlap (that's the strongest copy)
3. **Refine** — Have each expert critique from their angle
4. **Finalize** — One voice emerges, not four

## Expert Prompts

When activating each expert, ask them to address:

**Solana Mobile Expert:**
```
As the Solana Mobile Expert, answer: What do Saga/Seeker builders actually need?
What keeps them up at night? What would make them say "finally"?
```

**Product Positioning Expert:**
```
As the Product Positioning Expert, answer: What is the one positioning that makes this
the obvious choice? What category are we creating or conquering?
```

**B2B SaaS Copywriter:**
```
As the B2B SaaS Copywriter, answer: What's the headline that converts?
What's the pricing strategy? What's the conversion-blocking mistake to avoid?
```

**Web3 Native Marketer:**
```
As the Web3 Native Marketer, answer: Does this sound like it's from the community?
What on-chain proof points work best? What triggers skepticism?
```

## Output

The final copy should pass all four expert tests:
- [ ] Solana Mobile Expert approves
- [ ] Product Positioning Expert approves
- [ ] B2B SaaS Copywriter approves
- [ ] Web3 Native Marketer approves
