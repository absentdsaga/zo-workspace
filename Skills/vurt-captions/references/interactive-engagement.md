# Interactive Engagement Reference

Pulled from VURT Social Playbook v1.3. Apply this on every caption brief where IG Stories or TT/FB engagement is in scope.

## Story Stack — 4-Frame Replies-First Sequence (Default)

Updated 2026-04-29. Old 1/2/3-story tiers retired — they put reshares in slot 1 and starved the algo. IG Stories reach is reply-weighted; one reply lifts the next 4-5 frames. Every clip post runs the 4-frame pattern below unless it's a UGC/press repost (1-frame light touch still applies for those).

**Default (every clip post + every premiere):**

1. **Frame 1 — Native hook.** Quoted-dialogue on text-on-black, OR a held cliffhanger frame with overlay copy. **Never a pure reshare in slot 1** — pure reshares get throttled by the algo.
2. **Frame 2 — Reshare the Reel** with NEW overlay copy (different angle than the Reel caption). Don't reuse the line.
3. **Frame 3 — Replies-flywheel sticker** (poll / question / quiz / slider / Add Yours / DM sticker). Match scene type via the sticker picker below. This is the most important frame — never skip.
4. **Frame 4 — Link sticker on a cliffhanger frame** with cliffhanger CTA copy tied to the unresolved stake. Generic "Watch Now" is dead.

**1-frame light touch** — UGC reposts, filmmaker testimonials, press reposts: reshare + link sticker only.

**Daily cadence floor:** 3-4 stories minimum per day. The 4-frame sequence counts toward this. Daily presence is non-negotiable for accounts under 90 days old — Stories sort by recency in the top bar.

**Native-vs-reshare ratio:** Target 2 native frames for every 1 reshare across the day. Pure-reshare days get throttled.

**Tag everything:** Cast handles, location, sound stickers on every frame where it fits. Each tag is a distribution lever and a notification (which generates replies).

**30-day targets (current vs goal):**
- Reach per frame: 200+ (baseline ~13)
- Replies per 4-frame sequence: 5-10
- Link sticker CTR: 1%+

## IG Sticker Picker

| If the scene is… | Use this sticker | Example |
| --- | --- | --- |
| Choice / dilemma | Poll (binary) | "Pick it up / Hand her the phone." |
| Emotional reaction | Slider (emoji) | "How did this scene hit?" 🔥 |
| Character trait moment | This or That quiz | "Red flag or just dumb?" |
| Theory-bait moment | Question sticker | "What does she do next?" |
| Relatable scene | Add Yours | "Reply with the line that hit hardest." |
| BTS / UGC clip | Q&A or DM sticker | "Ask the cast anything — DM us." |
| Premiere countdown | Countdown | "Premieres Wed — set the reminder." |
| Music-driven clip | Music sticker + emoji slider | Drop track + slider for vibe. |

Rules:
- One sticker per frame.
- **Frame 3 is the replies-flywheel slot.** Never put the interactive sticker in Frame 1.
- Polls must be binary and dramatic — no neutral middle option.
- Reply to every DM and story reply within 24 hours. Replies generate algo signal AND give you content to reshare on the next day's stack.
- Run one Add Yours template per active title per week — highest-virality story mechanic on IG.

## TikTok Engagement Plays

1. End caption with a question.
2. Pin a comment with a *different* question.
3. Reply to commenters with video replies (free reach).
4. Use Q&A profile feature.
5. Allow Stitches/Duets when scene invites reactions.

## Facebook Engagement Plays

1. Question in caption.
2. Tag relevant Pages.
3. Reply by tagging commenters back.
4. Share to relevant FB Groups (only ones VURT belongs to).

## YouTube Engagement Plays

1. Pinned comment with a question.
2. End screen — Subscribe + most recent Short.
3. Reply to early commenters within the first hour.
4. Use Community posts for between-clip polls (when threshold hit).

## What to Never Skip

- Every clip post runs the 4-frame replies-first sequence (Frame 3 is the replies sticker, never optional).
- Frame 1 is always native (quoted-dialogue still or cliffhanger frame), never a pure reshare.
- Frame 4 link sticker copy ties to the unresolved stake of THIS clip — no generic "Watch Now."
- Captions end with destination OR question.
- Replies/comments/DMs answered within 24 hours.
- No repeat poll/question templates within a week.
- 3-4 stories minimum per day. 2 native for every 1 reshare.
