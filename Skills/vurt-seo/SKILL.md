# VURT SEO & AI Search Optimization Skill

## Purpose
Drive VURT's discoverability across Google Search, AI search engines (ChatGPT, Perplexity, Google AI Overviews, Claude), App Store search, and social search (TikTok, YouTube, Instagram). The goal is to make VURT the #1 result when anyone searches for vertical drama, Black streaming, micro-series, or culture-driven entertainment.

## Dual-Track Strategy: Google + AI Search
Traditional SEO and AI search optimization (AIO/GEO) require overlapping but distinct approaches. We optimize for both simultaneously.

### Google wants:
- Crawlable, indexable pages with unique meta tags
- Structured data (Schema.org)
- Fast load times, mobile-first
- Quality backlinks from authoritative domains
- Fresh, original content
- No dead links

### AI search engines want:
- Clear, well-structured content that can be cited
- Authoritative mentions across the web (Wikipedia, press, blogs, forums)
- Clean HTML with semantic markup
- No dead links (even more critical than Google — AI crawlers abandon domains with broken links)
- Consistent entity information (name, description, facts) across all sources
- Rich, unique descriptions (not repetitive boilerplate)

## Expert Input (Dioni's SEO advisor)
Key directives from experienced SEO professional:
1. **AI search is the priority** — more people searching via AI, optimize for that
2. **Rich text + structured data** — give crawlers order, detail, indexable content
3. **Image SEO**: EXIF data, descriptive filenames (`vurt-[what-it-is].jpg`), unique metadata per asset
4. **No repetitive metadata** — each asset description must be unique to what the file actually is + relevant keywords
5. **Long-tail keywords** — underused, less competitive, high-intent. Think like the crawler.
6. **Dead links kill AI rankings** — audit and fix immediately
7. **Backlink quality > quantity** — Web 2.0 blogs (Medium), X articles/threads, social threads are the new SEO backlink strategy
8. **Popular + less competitive keywords** — find the gaps competitors aren't filling

## Core Workstreams

### 1. Technical SEO (myvurt.com)
- Fix Angular SPA rendering for crawlers (SSR/pre-rendering)
- Unique title + meta description per page
- Open Graph + Twitter Card tags on every page
- Schema.org markup (Organization, WebSite, TVSeries, Episode, VideoObject)
- Canonical URLs
- robots.txt + sitemap.xml
- Dead link audit and fix
- Image alt text + descriptive filenames
- Core Web Vitals optimization

### 2. Keyword Strategy
- Primary clusters: vertical streaming, micro-drama, Black streaming, culture-driven entertainment
- Long-tail targets: "free Black drama series app", "vertical video streaming platform", "micro-drama like ReelShort but free", "Black indie filmmaker streaming"
- Search intent mapping: informational, navigational, transactional
- Competitor keyword gap analysis (ReelShort, DramaBox, Tubi, BET+)

### 3. Content SEO (Off-Site Authority Building)
- Medium publication for VURT (long-form articles on vertical cinema, creator economy, Black entertainment)
- X/Twitter threads as SEO artifacts (Google indexes X threads)
- LinkedIn articles (for B2B/investor/industry visibility)
- Reddit engagement in r/streaming, r/BlackFilm, r/filmmakers, r/shortfilm
- Press coverage amplification (TechCrunch backlink is gold — build more)
- Creator spotlight blogs (each creator = a unique indexable page/article)

### 4. Image & Video SEO
- Filename convention: `vurt-[series-name]-[descriptor]-[episode].jpg`
- Alt text: unique, descriptive, keyword-rich but natural
- EXIF data: embed copyright, description, keywords in image metadata
- Open Graph images: custom per page/series (not generic VURT logo)
- Video schema markup for every series/episode
- Thumbnail optimization for search results

### 5. App Store Optimization (ASO)
- Keyword-rich app title and subtitle
- Optimized app description with target keywords
- Screenshot alt text and captions
- Review solicitation strategy
- Deep linking (app content indexable by Google)

### 6. AI Search Optimization (AIO/GEO)
- Ensure VURT appears in AI-generated answers about streaming, micro-drama, vertical video
- Build web presence that AI can cite: Wikipedia mention, Crunchbase, press coverage
- Consistent entity description across all platforms
- FAQ schema (AI loves pulling from FAQ structured data)
- "People Also Ask" targeting
