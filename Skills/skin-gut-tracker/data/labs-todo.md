## Tier 1 — do first

- [ ] **Nickel patch test (T.R.U.E. test)** — dermatologist, 48hr read. Confirms active nickel sensitization.
- [ ] **CBC with differential** — eosinophils, basophils (allergy/mast-cell-adjacent signal)
- [ ] **Anti-tTG IgA + total IgA** — celiac rule-out. Must be eating gluten 6+ weeks before test.
- [ ] **Vitamin D, 25-OH**
- [ ] **Vitamin B12**
- [ ] **Ferritin**
- [ ] **Zinc (serum)**
- [ ] **CRP (high-sensitivity)**
- [ ] **Homocysteine**

## Tier 2 — if Tier 1 doesn't explain it

- [ ] **Serum tryptase** (baseline — done fasting, not during flare)
- [ ] **DAO (diamine oxidase) level** — Mosaic Diagnostics or KBMO
- [ ] **H. pylori breath test** (or stool antigen)
- [ ] **SIBO breath test** (lactulose, 3-hour)
- [ ] **GI-MAP stool test** (Diagnostic Solutions — order via Rupa Health or functional med doc)

## How to order without a doctor (US)

- **Rupa Health** (rupahealth.com) — wholesale access to most of the above
- **Ulta Lab Tests / Quest Direct / Labcorp OnDemand** — for standard blood panels
- **Mosaic Diagnostics** — DAO, organic acids
- **Diagnostic Solutions** — GI-MAP (requires practitioner)

## Derm referral

For patch test + to discuss:
- Dupilumab (Dupixent) for refractory pompholyx
- Disulfiram protocol (chelates nickel — specialist only)
- Topical tacrolimus as steroid-sparing alternative
