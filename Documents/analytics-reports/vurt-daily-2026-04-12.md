## Insights & Actions

1. **▼ TODAY'S KEY SIGNAL — Avg session duration: -38% day-over-day** (6m 14s → 3m 50s)

2. **Platform Health Score: 35/100 (D)**
   Growth: 0/25 | Engagement: 24/25 | Retention: 3/25 | Traffic Quality: 8/25
   **How it's scored:** Growth (0/25): WoW user growth 15% — 500%+ = 25/25 | Engagement (24/25): 24.2% engagement rate — 25%+ = 25/25 | Retention (3/25): GA4 not reporting new vs returning — scored 3 (data gap) | Traffic Quality (8/25): 17% organic/direct traffic — 50%+ = 25/25
   *For a 4-week-old platform, focus on Engagement and Retention — Growth can be bought, stickiness can't.*

3. **Organic signal:** 17% of traffic is organic/direct, but it drives 65% engagement — 4x more valuable per session than paid. Best channel: Organic Search at 74% engagement, 77m 13s avg. **Action:** Prioritize the SEO fixes from the dev call — auto-populated sitemap, SSR/pre-rendering for Angular, unique episode URLs, and per-page meta tags. Each fix directly grows this high-value channel.

4. **The app is the product, the web is the funnel:** App engagement is 89% with 69m 8s sessions. Web is 22% with 1m 32s. That's a 4x gap — 12.2K web users vs 308 on app. **Action:** Measure web-to-app conversion rate. Ensure app download CTAs are prominent on mobile web (above the fold, post-episode). Given the 4x engagement difference, every percentage point of web→app conversion meaningfully grows engaged users.

5. **Android users are power viewers:** 127m 9s avg sessions vs 11m 7s on iOS. Only 139 Android users vs 169 iOS — but they watch longer. Store ratings: iOS: 5.0★ (13 ratings), Android: 0 ratings.

6. **Retention data gap:** GA4 isn't splitting new vs returning users cleanly. This may be a data stream configuration issue — worth flagging to enveu's dev team.

7. **Low session depth:** 1.2 sessions per user means almost nobody comes back within the same week. For a content platform, target is 2.0+. **This is the leaky bucket:** paid ads bring people in, they watch once (or bounce), and don't return. Solve this before scaling ad spend further.

8. **Tracking gap:** 314 users spent avg 10m 57s on content GA4 labels '(not set)'. This means screen names aren't being passed in GA4 screen_view events for these views. **Action for dev team:** Ensure every screen_view event includes the series/episode title so this engagement can be properly attributed.
