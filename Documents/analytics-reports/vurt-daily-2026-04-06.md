### Methodology
- **Data source:** Google Analytics 4 (Property ID 518738893, myvurt.com)
- **Engagement Rate:** % of sessions lasting >10s, with 2+ page views, or a key event (GA4 standard definition)
- **Bounce Rate:** % of sessions that were NOT engaged (inverse of engagement rate — session lasted <10s, had only 1 page view, and no key events)
- **Daily Snapshot freshness:** Automatically uses yesterday's data when GA4 processing is complete (engaged sessions >= 30% of 7-day avg). Falls back to 2-days-ago when data is still processing.
- **Social media data:** Instagram via Meta Graph API (IG Business Account 17841479978232203), YouTube/TikTok/X via public profile scraping
- **All numbers are pulled directly from GA4 APIs** — no manual adjustments or estimates except where explicitly labeled as projections
- **Ad revenue projections** use verified AVOD CPM benchmarks ($15-25, Adwave 2025) and are scaling models, not forecasts
